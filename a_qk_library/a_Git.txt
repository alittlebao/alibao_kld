设置你的用户名称与邮件地址
$ git config --global user.name "John Doe"
$ git config --global user.email johndoe@example.com

配置默认文本编辑器
$ git config --global core.editor vim
$ git config --global core.editor emacs
$ git config --global core.editor " 'C:/Program Files/Notepad++/notepad++.exe' -multiInst -nosession"
$ git config --global core.editor " 'C:/Program Files (x86)/Notepad++/notepad++.exe' -multiInst -nosession"

$ git config --list

$ git remote rename origin Eric

$ git remote add origin (刚才复制的远程库的SSH)

强制push：
$ git push -u origin master -f

撤销commit
git reset Cmixed ：此为默认方式，不带任何参数的git reset，即时这种方式，它回退到某个版本，只保留源码，回退commit和index信息。
git reset Csoft ：回退到某个版本，只回退了commit的信息，不会恢复到index file一级。如果还要提交，直接commit即可。
git reset Chard ：彻底回退到某个版本，本地的源码也会变为上一个版本的内容。
